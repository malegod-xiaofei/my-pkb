- Linux执行脚本的几种方式
  - 相对路径：切换到shell脚本所在的目录，执行shell脚本  ./hello.sh
  - 绝对路径：写全脚本的路径  /data/shell/hello.sh
  - 直接使用bash或sh来执行，source执行，影响当前对话窗口，nohup
  - 在当前shell环境中执行bash shell脚本
- Linux的文件权限
  - 4 - 2 - 1 （读-写-可执行）
  - 文件所有者的-同组其他人的-其他人的
  - 读写改权限
  - 赋予权限 chmod  600 （对文件加中的所有文件赋予权限要加大写\R）
- nohup加&：表示在后台运行
- shell脚本看文本文件的内容：
  - cat
  - vi  vim
  - more
- sort -u data_name.txt、实现去重
  - sort -u data_name.txt | wc -l 、实现统计数量
- 查看文本文件的前十行
  - cat filename | tail -n +3000 | head -n 1000  (从第3000行开始，显示1000行。即显示3000~3999行 )
  - cat filename | head -n 3000 | tail -n +1000 (显示1000到3000行)
  - sed -n '5,10p' filename  这样你就可以值查看文件的第5行到第10行
- 动态监控文件内容
  - tail -f file
