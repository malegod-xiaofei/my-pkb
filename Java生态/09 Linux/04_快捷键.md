- 查找和替换
  - 查找
    - 向下查找：输入/关键字，然后按回车键，vi会从当前光标位置开始向下查找第一个匹配的关键字。按 n 继续查找第一个匹配项
    - 向上查找：输入？关键字，按回车键，这将从当前位置向上查找匹配的关键字，按n查找上一个关键字
  - 替换
    - 在 vi 的末行模式，按：进入，可以用 s 进行替换操作，基本语法如下：
      - 替换当前行首次出现：:s/旧文本/新文本 将当前行中首次出现的旧文本替换为新文本。
      - 全局替换当前行：:s/旧文本/新文本/g g标志表示全局替换，即替换当前行中所有出现的旧文本。
      - 替换并确认每个替换：:s/旧文本/新文本/c c标志要求在每次替换前进行确认。
      - 替换整个文件中的文本：
        - 首先，在正常模式下按:进入末行模式，
        - 输入%s/旧文本/新文本/g 来全局替换整个文件中所有匹配的旧文本为新文本。
        - 如果希望在替换前确认每个替换，可以使用%s/旧文本/新文本/gc。
    - 例如，如果你想在当前文件中将所有出现的"example"替换为"sample"，并且在每次替换之前都需要确认，你可以使用命令:%s/example/sample/gc。
- 复制->Ctrl+Insert，粘贴->Shift+Insert
- Vi编辑模式下，按Esc退出编辑模式，光标处于哪一行，按dd快速删除此行
