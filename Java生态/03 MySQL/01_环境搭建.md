- 1、安装mysql
  - 1.解压到安装目录下
  - 2.配置环境变量
    - MYSQL_HOME
    - D:\Software\mysql-5.7.23-winx64(自己的安装路径)
    - path路径下添加%MYSQL_HOME%\bin;
  - 3.添加my.ini文件到安装目录文件下，将dir改成自己的路径
[mysqld]

port = 3306

basedir=D:\Software\mysql-5.7.23-winx64

datadir=D:\Software\mysql-5.7.23-winx64\data

max_connections=200

character-set-server=utf8

default-storage-engine=INNODB

sql_mode=NO_ENGINE_SUBSTITUTION,STRICT_TRANS_TABLES

skip-grant-tables

[mysql]

default-character-set=utf8

  - 4.运行cmd到mysql根目录bin下(以管理员身份运行)
    - 运行
mysqld -install，看到Service successfully installed.说明安装成功

  - 5.mysqld --initialize-insecure --user=mysql
    - 执行后回产生一个data文件夹，对应的路径就是my.ini里面设置的路径
  - 6.执行以下命令以启动mysql服务 命令：net start mysql
  - 7.
首次完成上面的安装，用户名为 root 密码是空的，即在bin 的路径下，输入 mysql -uroot -p 后在密码处回车即可进入，现在需要改密码的可以按照下面的步骤实现：

    - 1)
输入use mysql命令，进入mysql数据库

    - 2)修改密码：update user set authentication_string=password('新密码') where user='root';
    - 执行成功后，输入
flush PRIVILEGES;清空下，退出重新登录，就可以了。以上是mysql5.7.23安装教程，其他版本可能有些许出入。
