# 1 1.介绍下echarts

- 组件的基本概念
  - 百度开源的一个图表类的统一的开发框架，通过他做图表，什么饼状图，柱状图都可以用，包括3D都行
- 组件的特点
  - 优点：背景很强大，百度的技术简单易用，功能强大
  - 缺点：在一些图层的显示上啊有一些小bug
- 应用场景
  - 几乎能够满足绝大部分公司图表可视化需求，移动端和pc端都可以用它
