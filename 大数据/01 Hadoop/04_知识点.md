- 大数据开发目录规范
  - Api 文档区
  - Cache 软件缓存路径
  - Code 代码区
  - Configure 工具区(win、mac)
  - Datas 数据区
  - Documents 文档
  - Download 软件下载目录
  - Module 根目录，存放安装包（压缩版文件）
  - Software 软件安装目录
  - Tools 工具区（linux）
  - Work 工作区代码文件
```java
mkdir Api;

mkdir Cache;

mkdir Code;

mkdir Configure;

mkdir Datas;

mkdir Documents;

mkdir Download;

mkdir Module;

mkdir Software;

mkdir Tools;

mkdir Work;
```
- 启动集群
  - /opt/module/hadoop-3.1.3
  - sbin/start-dfs.sh
- hadoop的几大模块
  - core-site.xml(工具模块)
  - hdfs-site.xml(数据存储模块)
  - mapred-site.xml(数据处理模块)
  - yarn-site.xml(作业调度+资源管理平台)
- 环境准备
  - 首先需要进行目录规划：/opt
  - 存储数据的目录：mkdir datas
  - 存储软件的目录：mkdir modules
  - 存储软件包目录：mkdir software
  - 存储工具的目录：mkdir tools
