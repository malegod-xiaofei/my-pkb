- git小游戏
- git上Push代码报错：Failed to connect to 127.0.0.1 port 8888: Connection refused
  - 解决办法：
    - 查看代理设置

```bash
git config --get http.proxy
git config --get https.proxy
```

    - 如果你发现代理配置有误，你可以用以下命令修改或删除代理设置：

```bash
git config --global http.proxy <http://your-proxy-url:port>
git config --global https.proxy <https://your-proxy-url:port>
```

    - 若要移除代理设置，使用：

```bash
git config --global --unset http.proxy
git config --global --unset https.proxy
```
