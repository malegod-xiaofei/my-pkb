# 1. 基础语法

| 文本编辑内容        | markdown使用方法      | 示例                                 |
| ------------- | ----------------- | ---------------------------------- |
| 链接            | 两个中括号             | `[[A笔记]]`                          |
| 链接名字显示其他文字    | 两个中括号 + \|        | `[[A笔记`\|`新名字]]`                   |
| 链接A笔记中的某个段落   | 两个中括号 + #         | `[[A笔记#标题]]`                       |
| 链接A笔记中的某句话    | 两个中括号 + ^         | `[[A笔记^某句话]]`                      |
| 把A笔记嵌入到当前笔记中  | ! + 两个中括号         | `![[A笔记]]`                         |
| 外部链接          | `[链接名称]`(地址)      | `[google](https://www.google.com)` |
| 一级标题          | # + 空格            | # 标题1                              |
| 二级标题          | ## + 空格           | ## 标题2                             |
| 高亮            | 两个 ==             | `==文字==`                           |
| 加粗(Ctrl + B)  | 两个 ** 或 _ _       | `**文字** __文字__ `                   |
| 斜体            | 单个 *              | `*文字*`                             |
| 删除线           | 两个 ~~             | `~~文字`                             |
| 无序列表          | - + 空格            | `- 文字`                             |
| 选择框(Ctrl + L) | `- + 空格 + [ ]+空格` | `- [] `                            |
| 选择框勾选         | `- + 空格 + [x]+空格` | `- [x]`                            |
| 引用            | > + 空格            | >                                  |
| 标注            | 右键 -> 插入 -> 标注    | `> [!bug]- `                       |
| 脚注(阅读模式)      | `^ + [ 脚注内容 ]`    | `^[1]`                             |
| 行内代码          | 两个 ``             | `文字`                               |
| 代码块           | 三个 ```            | ``````                             |
