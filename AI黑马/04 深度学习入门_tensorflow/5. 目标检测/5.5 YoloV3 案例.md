# 2.TFrecord⽂件
该案例中我们依然使⽤VOC数据集来进⾏⽬标检测，不同的是我们要利⽤ tfrecord⽂件来存储和读取数据，⾸先来看⼀下tfrecord⽂件的相关内容。

为什么要使⽤tfrecord⽂件？

+ TFRecord是Google官⽅推荐使⽤的数据格式化存储⼯具，为 TensorFlow量身打造的。
+ TFRecord规范了数据的读写⽅式，数据读取和处理的效率都会得到 显著的提⾼。

## 2.1 什么是TFrecord⽂件
TFRecord 是Google官⽅推荐的⼀种数据格式，是Google专⻔为 TensorFlow设计的⼀种数据格式，利⽤这种⽅式存储数据可以使其与⽹络 架构更适配。TFRecord是⼀种⼆进制⽂件，其能更好的利⽤内存，与 csv,hdf5⽂件是类似的。

TFRecord的⽂件的内容如下图所示：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742701435622-dcd70785-6fed-4594-a0be-718849ff1742.png)

TFRecord内部包含多个tf.train.Example，⼀般来说对应⼀个图像数据， 在⼀个Example消息体中包含了⼀系列的tf.train.feature属性，⽽ 每⼀个 feature是⼀个key-value的键值对,key是特征名称，value是特征值。

TFRecord 并⾮是TensorFlow唯⼀⽀持的数据格式，也可以使⽤CSV或⽂本等其他格式，但是对于TensorFlow来说，TFRecord 是最友好的，最⽅便的，⽽且tensorflow也提供了丰富的API帮助我们轻松的创建和获取 TFRecord⽂件。

## <font style="color:black;">4.2 正负样本的设定</font>
<font style="color:black;">在上述的</font><font style="color:black;">loss</font><font style="color:black;">计算中，负责进⾏⽬标预测的</font><font style="color:black;">anchor</font><font style="color:black;">就是正样本，⽽不负责 进⾏⽬标预测的就是负样本，也就是背景，那在这⾥我们是如何设置正负 样本的呢？如下图所示：</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742716266323-2ced1dd6-4d28-4657-b71f-55063f4f4d49.png)

+ **<font style="color:black;">正样本</font>**<font style="color:black;">：⾸先计算⽬标中⼼点落在哪个grid上，然后计算这个grid对 应的3个先验框（anchor）和⽬标真实位置的IOU值，取IOU值最⼤的 先验框和⽬标匹配。那么该anchor就负责预测这个⽬标，那这个 anchor就作为正样本，将其置信度设为1，其他的⽬标值根据标注信 息设置。</font>
+ **<font style="color:black;">负样本</font>**<font style="color:black;">：所有不是正样本的anchor都是负样本，将其置信度设为0， 参与损失计算，其它的值不参与损失计算，默认为0。</font>

<font style="color:black;">对于每⼀个</font><font style="color:black;">anchor</font><font style="color:black;">我们都要</font><font style="color:black;">4+1+8</font><font style="color:black;">0</font><font style="color:black;">维的⽬标值，其中前</font><font style="color:black;">4</font><font style="color:black;">维是坐标值，正 样本是</font><font style="color:black;">GT</font><font style="color:black;">的</font><font style="color:black;">bbox</font><font style="color:black;">框的值，第</font><font style="color:black;">5</font><font style="color:black;">维是置信度，正样本设置为</font><font style="color:black;">1</font><font style="color:black;">，负样本设置 为</font><font style="color:black;">0</font><font style="color:black;">，最后的</font><font style="color:black;">80</font><font style="color:black;">是类别数，正样本对应的类别设置为</font><font style="color:black;">1</font><font style="color:black;">，其余为</font><font style="color:black;">0</font><font style="color:black;">，若使⽤ </font><font style="color:black;">voc</font><font style="color:black;">数据集类别数是</font><font style="color:black;">20</font><font style="color:black;">。</font>

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742716323459-b108fb51-e3eb-4ac1-9169-337e35935f02.png)

```python
trainset = dataset("dataset/voc_train.tfrecords", batch_size)
```

**<font style="color:black;">4.3.2</font>**<font style="color:black;">加载模型</font>

<font style="color:black;">将在yoloV3模型和损失函数的计算进⾏实例化：</font>

```python
# V3模型的实例化，指定输⼊图像的⼤⼩，即⽬标检测的类别个数
yolov3 = YOLOv3((416, 416, 3,), 20)
yolov3_loss = Loss((416, 416, 3), 20)
```

**<font style="color:black;">4.3.3</font>**<font style="color:black;">模型训练</font>

<font style="color:black;">模型训练也就是要使⽤损失函数，进⾏反向传播，利⽤优化器进⾏参数更 新，训练的流程是：</font>

<font style="color:black;">1、指定优化器：在这⾥我们使⽤加动量的SGD⽅法</font>

<font style="color:black;">2、设置epoch，进⾏遍历获取batch数据送⼊⽹络中进⾏预测</font>

<font style="color:black;">3、计算损失函数，使⽤反向传播更新参数，我们使⽤tf.GradientTape实 现：</font>

+ <font style="color:black;">定义上下⽂环境：tf.GradientTape</font>
+ <font style="color:black;">计算损失函数loss</font>
+ <font style="color:black;">使⽤tape.gradient(loss,model.trainable_variables)⾃动 计算梯度，loss是损失结果，trainable_variables为所有需要训练的变量。</font>
+ <font style="color:black;">使⽤ optimizer.apply_gradients(zip(grads,model.trainable_variables))⾃动更新模型参数，zip(grads,trainable_variables)将梯度和参数关联起来，然后apply_gradients会⾃动的利⽤梯度对参数进⾏更新。</font>

<font style="color:black;">接下来我们按照这个流程完成模型训练，并保存模型训练结果。</font>

```python
# 1、定义优化⽅法
optimizer = tf.keras.optimizers.SGD(0.1, 0.9)
# 2.设置epoch，获取batch数据送⼊⽹络中进⾏预测
for epoch in range(300):
    loss_history = []
    # 遍历每⼀个batch的图像和⽬标值，进⾏更新
    for (batch, inputs) in enumerate(trainset):
        images, labels = inputs
        # 3.计算损失函数，使⽤反向传播更新参数 
        # 3.1 定义上下⽂环境
        with tf.GradientTape() as tape:
            # 3.2 将图像送⼊⽹络中
            outputs = yolov3(images)
            # 3.3 计算损失函数
            loss = yolov3_loss([*outputs, *labels])
            # 3.4 计算梯度
            grads = tape.gradient(loss, yolov3.trainable_variables)
            # 3.5 梯度更新
            optimizer.apply_gradients(zip(grads, yolov3.trainable_variables))
            # 3.6 打印信息
            info = 'epoch: %d, batch: %d ,loss: %f' % (epoch, batch, np.mean(loss_history))
            print(info)
            loss_history.append(loss.numpy())
yolov3.save('yolov3.h5')
```
