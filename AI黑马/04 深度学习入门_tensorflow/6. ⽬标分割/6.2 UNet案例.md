### 1.2.3 构建数据集⽣成器
利⽤keras.utils.Sequence构建图像⽣成器来读取数据，每个Sequence必 须实现 getitem 和 len ⽅法，通过 getitem 应返回完整的批次， Sequence是进⾏多处理的更安全⽅法。这种结构保证了⽹络在每个时间 段的每个样本上只会训练⼀次。主要实现3个⽅法；init,len和getitem即可。

```python
from tensorflow import keras
import numpy as np
from tensorflow.keras.preprocessing.image import load_img


# 数据集获取：
class OxfordPets(keras.utils.Sequence):
    # 在__init__⽅法中指定batch_size,img_size,input_img_paths
    def init(self, batch_size, img_size, input_img_path, target_img_paths):
        self.batch_size = batch_size  # 批量⼤⼩
        self.img_size = img_size  # 图像⼤⼩
        self.input_img_paths = input_img_paths  # 输⼊图像路径
        self.target_img_paths = target_img_paths  # 标注图像路径

    def len(self):  # 计算迭代次数
        return len(self.target_img_paths) // self.batch_siz

    def getitem(self, idx):
        """
        获取每⼀个batch数据
        """
        i = idx * self.batch_size
        # 获取输⼊的图像数据
        batch_input_img_paths = self.input_img_paths[i: i + self.batch_size]
        # 获取标签数据
        batch_target_img_paths = self.target_img_paths[i: i + self.batch_size]
        # 构建特征值数据：获取图像数据中每个像素的数据存储在x中
        x = np.zeros((batch_size,) + self.img_size + (3,), dtype="float32")
        for j, path in enumerate(batch_input_img_paths):
            img = load_img(path, target_size=self.img_size)
            x[j] = img
        # 构建⽬标值数据：获取标注图像中每个像素中的数据存在y中
        y = np.zeros((batch_size,) + self.img_size + (1,), dtype="uint8")
        for j, path in enumerate(batch_target_img_paths):
            img = load_img(path, target_size=self.img_size, color_mode="grayscale")
            y[j] = np.expand_dims(img, 2)
        return x, y
```

### 1.3.1 编码部分
编码部分的特点是：

+ 架构中含有着⼀种重复结构，每次重复中都有2个 3 x 3卷积层、⾮线 性ReLU层和⼀个 2 x 2 max pooling层（stride为2）。

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742721436446-4030908d-346a-418c-b76f-2e396e24c533.png)

+ 每⼀次下采样后我们都把特征通道的数量加倍
+ 每次重复都有两个输出：⼀个⽤于编码部分进⾏特征提取，⼀个⽤于 解码部分的特征融合

构建的代码如下所示：

```python
# 输⼊：输⼊张量，卷积核个数
def downsampling_block(input_tensor, filters):
    # 输⼊层
    x = Conv2D(filters, kernel_size=(3, 3), padding='same')(input_tensor)
    # BN层
    x = BatchNormalization()(x)
    # 激活函数
    x = Activation('relu')(x)
    # 卷积层
    x = Conv2D(filters, kernel_size=(3, 3), padding="same")(x)
    # BN层
    x = BatchNormalization()(x)
    # 激活层
    x = Activation('relu')(x)
    # 返回的是池化后的值和激活未池化的值，激活后未池化的值⽤于解码部分
    return MaxPooling2D(pool_size=(2, 2))(x), x
```

### 1.3.2 解码部分
解码部分也使⽤了重复模块：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742721574239-249cf36f-e8ea-4c9d-ac87-bd29d2589818.png)

+ 每⼀个模块有两个输⼊：⼀个是编码阶段的特征图，⼀个是解码部分 的特征图
+ 每⼀步都⾸先使⽤反卷积(up-convolution)，每次使⽤反卷积都将特征 通道数量减半，特征图⼤⼩加倍。（图中绿箭头）
+ 反卷积过后，将反卷积的结果与编码部分中对应步骤的特征图拼接起 来。（⽩/蓝块）
+ 编码部分中的特征图尺⼨稍⼤，将其修剪过后进⾏拼接。（左边深蓝 虚线）
+ 对拼接后的map再进⾏2次3 x 3的卷积。（右侧蓝箭头）

编码实现如下：

```python
# 输⼊：输⼊张量，特征融合的张量，卷积核个数
def upsampling_block(input_tensor, skip_tensor, filters):
    # 反卷积
    x = Conv2DTranspose(filters, kernel_size=(2, 2), strides=(2, 2), padding="same")(input_tensor)
    # 获取当前特征图的尺⼨
    _, x_height, x_width, _ = x.shape
    # 获取要融合的特征图的尺⼨
    _, s_height, s_width, _ = skip_tensor.shape
    # 获取特征图的⼤⼩差异
    h_crop = s_height - x_height
    w_crop = s_width - x_width
    # 若特征图⼤⼩相同不进⾏裁剪
    if h_crop == 0 and w_crop == 0:
        y = skip_tensor
    # 若特征图⼤⼩不同，使级联时像素⼤⼩⼀致
    else:
        # 获取特征图裁剪后的特征图的⼤⼩
        cropping = ((h_crop // 2, h_crop - h_crop // 2), (w_crop // 2, w_crop - w_crop // 2))
        # 特征图裁剪
        y = Cropping2D(cropping=cropping)(skip_tensor)
    # 特征融合
    x = Concatenate()([x, y])
    # 卷积
    x = Conv2D(filters, kernel_size=(3, 3), padding="same")(x)
    # BN层
    x = BatchNormalization()(x)
    # 激活层
    x = Activation('relu')(x)
    # 卷积层
    x = Conv2D(filters, kernel_size=(3, 3), padding="same")(x)
    # BN层
    x = BatchNormalization()(x)
    # 激活层
    x = Activation('relu')(x)
    return x
```

### 1.3.3 模型构建
将编码部分和解码部分组合⼀起，就可构建unet⽹络，在这⾥unet⽹络的 深度通过depth进⾏设置，并设置第⼀个编码模块的卷积核个数通过filter 进⾏设置，通过以下模块将编码和解码部分进⾏组合：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/25755350/1742721831794-69661671-fc9a-40d7-8734-4d95bafe09c8.png)

```python
# 使⽤3个深度构建unet⽹络
def unet(imagesize, classes, features=64, depth=3):
    # 定义输⼊数据
    inputs = keras.Input(shape=img_size + (3,))
    x = inputs
    # ⽤来存放进⾏特征融合的特征图
    skips = []
    # 构建编码部分
    for i in range(depth):
        x, x0 = downsampling_block(x, features)
        skips.append(x0)
        # 下采样过程中，深度增加，特征翻倍，即每次使⽤翻倍数⽬的滤波器
        features *= 2
    # 卷积
    x = Conv2D(filters=features, kernel_size=(3, 3), padding="same")(x)
    # BN层
    x = BatchNormalization()(x)
    # 激活
    x = Activation('relu')(x)
    # 卷积
    x = Conv2D(filters=features, kernel_size=(3, 3), padding="same")(x)
    # BN层
    x = BatchNormalization()(x)
    # 激活
    x = Activation('relu')(x)
    # 解码过程
    for i in reversed(range(depth)):
        # 深度增加，特征图通道减半
        features //= 2
        # 上采样
        x = upsampling_block(x, skips[i], features)
    # 卷积
    x = Conv2D(filters=classes, kernel_size=(1, 1), padding="same")(x)
    # 激活
    outputs = Activation('softmax')(x)
    # 模型定义
    model = keras.Model(inputs, outputs)
    return model
```

## 1.4 模型训练
### 1.4.1 数据集划分
数据集中的图像是按顺序进⾏存储的，在这⾥我们将数据集打乱后，验证 集的数量1000，剩余的为训练集，划分训练集和验证集：

```python
import random

# 将数据集划分为训练集和验证集，其中验证集的数量设为1000
val_samples = 1000
```
