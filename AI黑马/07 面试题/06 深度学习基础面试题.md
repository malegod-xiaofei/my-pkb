# 06 深度学习基础面试题

> 参考笔记：`AI黑马/04 深度学习入门_tensorflow`、`AI黑马/05 NLP文档/01_PyTorch`。
>
> 权威参考：TensorFlow Guide、TensorFlow Keras Guide、Deep Learning Book、CS231n。

## 1. 神经网络的基本组成是什么？

**答案要点：**

神经网络通常由输入层、隐藏层和输出层组成。每一层包含若干神经元，通过权重和偏置进行线性变换，再经过激活函数引入非线性。

核心要素：

- 参数：权重和偏置。
- 前向传播：计算预测结果。
- 损失函数：衡量预测和真实标签的差异。
- 反向传播：计算梯度。
- 优化器：根据梯度更新参数。

## 2. 为什么神经网络需要激活函数？

**答案要点：**

激活函数用于引入非线性。如果没有激活函数，多层线性变换仍然等价于单层线性变换，模型表达能力有限。

常见激活函数：

- Sigmoid：输出 0 到 1，容易梯度消失。
- Tanh：输出 -1 到 1，也可能梯度消失。
- ReLU：简单高效，缓解梯度消失，但可能出现神经元死亡。
- Leaky ReLU、GELU：常见改进形式。

## 3. 前向传播和反向传播分别做什么？

**答案要点：**

- 前向传播：输入数据经过网络逐层计算，得到预测值和损失。
- 反向传播：根据链式法则从损失函数开始向前计算各参数梯度。
- 优化器根据梯度更新参数，使损失逐步下降。

## 4. 常见损失函数有哪些？

**答案要点：**

- 回归任务：MSE、MAE、Huber Loss。
- 二分类任务：Binary Cross Entropy。
- 多分类任务：Cross Entropy。
- 序列任务：交叉熵、CTC Loss 等。
- 度量学习：Triplet Loss、Contrastive Loss。

选择损失函数时要匹配任务目标和输出形式。

## 5. SGD、Momentum、Adam 有什么区别？

**答案要点：**

- SGD：按小批量梯度更新参数，实现简单但可能震荡。
- Momentum：引入动量项，累积历史梯度方向，加速收敛并减少震荡。
- Adam：结合一阶矩和二阶矩自适应调整学习率，收敛快，默认常用。
- Adam 不一定总是泛化最好，实际中需要结合学习率、权重衰减等调参。

## 6. 什么是梯度消失和梯度爆炸？

**答案要点：**

- 梯度消失：反向传播中梯度逐层变小，前面层参数难以更新。
- 梯度爆炸：梯度逐层变大，导致参数更新不稳定甚至 NaN。

解决方法：

- 使用 ReLU、GELU 等激活函数。
- 合理初始化参数。
- BatchNorm 或 LayerNorm。
- 残差连接。
- 梯度裁剪。
- 选择合适学习率。

## 7. Dropout 的作用是什么？

**答案要点：**

Dropout 在训练时随机丢弃部分神经元，减少神经元之间的复杂共适应关系，从而缓解过拟合。

- 训练时启用 Dropout。
- 推理时关闭 Dropout，并使用完整网络。
- 常用于全连接层、部分卷积网络和 Transformer 中。

## 8. BatchNorm 的作用是什么？

**答案要点：**

BatchNorm 对小批量数据的中间激活做标准化，再通过可学习参数恢复表达能力。

作用：

- 稳定训练过程。
- 允许使用较大学习率。
- 缓解内部协变量偏移。
- 有一定正则化效果。

推理阶段使用训练过程中累计的均值和方差。

## 9. 正则化在深度学习中有哪些形式？

**答案要点：**

- L1/L2 权重正则化。
- Dropout。
- 数据增强。
- Early Stopping。
- BatchNorm。
- 权重共享。
- Label Smoothing。

目标都是降低过拟合，提高泛化能力。

## 10. 学习率过大或过小会怎样？

**答案要点：**

- 学习率过大：训练不稳定，损失震荡甚至发散。
- 学习率过小：收敛慢，可能陷入局部区域或训练不足。
- 常用学习率策略：阶梯衰减、余弦退火、Warmup、自适应优化器。

## 11. CNN 中卷积层的作用是什么？

**答案要点：**

卷积层通过局部连接和权重共享提取局部空间特征。

优势：

- 参数量比全连接层少。
- 能利用图像局部结构。
- 平移等变性较好。
- 多层卷积可以逐步提取边缘、纹理、形状和语义特征。

## 12. 卷积中的 stride、padding、kernel 分别是什么？

**答案要点：**

- kernel：卷积核大小，决定局部感受区域。
- stride：卷积核滑动步长，影响输出尺寸。
- padding：在输入边缘补零或补其他值，控制边界信息和输出尺寸。

输出尺寸常按输入尺寸、卷积核大小、padding 和 stride 计算。

## 13. 池化层有什么作用？

**答案要点：**

池化层用于降低特征图尺寸、减少计算量，并增强局部平移不变性。

常见方式：

- 最大池化：保留局部最强响应。
- 平均池化：保留局部平均信息。
- 全局平均池化：常用于替代全连接层，减少参数。

## 14. ResNet 为什么能训练很深的网络？

**答案要点：**

ResNet 引入残差连接，让网络学习残差映射而不是完整映射。

优势：

- 缓解梯度消失。
- 让信息和梯度可以跨层传播。
- 降低深层网络退化问题。
- 支持训练更深的 CNN。

## 15. TensorFlow / Keras 中 Sequential API 和 Functional API 有什么区别？

**答案要点：**

- Sequential API 适合简单的线性堆叠模型。
- Functional API 适合多输入、多输出、共享层、残差连接等复杂结构。
- Model subclassing 更灵活，适合自定义训练和复杂前向逻辑。

## 16. TensorFlow 中 `GradientTape` 的作用是什么？

**答案要点：**

`GradientTape` 用于记录前向计算过程，并自动计算变量相对于损失的梯度。

常见流程：

1. 在 tape 上下文中执行前向传播。
2. 计算 loss。
3. 调用 `tape.gradient(loss, variables)` 获取梯度。
4. 使用优化器更新参数。

## 17. PyTorch 中 `requires_grad` 和 `backward()` 的作用是什么？

**答案要点：**

- `requires_grad=True` 表示需要跟踪该张量的计算图并计算梯度。
- `backward()` 从标量损失开始反向传播，计算叶子张量的梯度。
- 梯度会累积到 `.grad` 中，训练时通常需要先 `optimizer.zero_grad()`。

## 18. `model.train()` 和 `model.eval()` 有什么区别？

**答案要点：**

- `model.train()` 启用训练模式，Dropout 和 BatchNorm 按训练逻辑工作。
- `model.eval()` 启用推理模式，Dropout 关闭，BatchNorm 使用统计均值和方差。
- 推理时还应使用 `torch.no_grad()` 或类似机制减少显存和计算开销。

## 19. 什么是 Early Stopping？

**答案要点：**

Early Stopping 在验证集指标长期不提升时提前停止训练，避免继续拟合训练集噪声。

需要关注：

- 监控指标。
- patience 容忍轮数。
- 是否保存验证集最优模型。

## 20. 如何判断深度学习模型训练是否正常？

**答案要点：**

可以观察：

- 训练损失是否下降。
- 验证损失是否同步下降。
- 是否出现过拟合或欠拟合。
- 梯度是否为 NaN 或异常大。
- 学习率是否合适。
- 数据和标签是否对齐。
- 指标是否明显优于随机基线。
