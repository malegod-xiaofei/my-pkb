# 07 计算机视觉与OpenCV面试题

> 参考笔记：`AI黑马/04.5 计算机视觉_OpenCV`、`AI黑马/04 深度学习入门_tensorflow` 中图像分类、目标检测和图像分割章节。
>
> 权威参考：OpenCV 官方文档、OpenCV-Python Tutorials、CS231n、Computer Vision: Algorithms and Applications。

## 1. OpenCV 中图像通常如何表示？

**答案要点：**

OpenCV Python 中图像通常表示为 NumPy `ndarray`。

- 灰度图通常 shape 为 `(H, W)`。
- 彩色图通常 shape 为 `(H, W, C)`。
- OpenCV 默认颜色通道顺序是 BGR，不是 RGB。
- 图像像素常用 `uint8`，取值范围 0 到 255。

## 2. RGB、BGR、HSV 有什么区别？

**答案要点：**

- RGB：按红、绿、蓝通道表示颜色。
- BGR：OpenCV 默认通道顺序。
- HSV：用色调、饱和度、明度表示颜色，更适合颜色分割和光照变化处理。
- 图像显示或与 Matplotlib 结合时要注意 BGR 与 RGB 转换。

## 3. 图像几何变换有哪些？

**答案要点：**

常见几何变换：

- 缩放。
- 平移。
- 旋转。
- 仿射变换。
- 透视变换。

几何变换通常需要插值，例如最近邻、双线性、双三次插值。

## 4. 腐蚀和膨胀有什么区别？

**答案要点：**

- 腐蚀会缩小前景区域，去除小白点或断开细小连接。
- 膨胀会扩大前景区域，填补小黑洞或连接断裂区域。
- 开运算是先腐蚀后膨胀，常用于去噪。
- 闭运算是先膨胀后腐蚀，常用于填充小孔。

## 5. 常见图像滤波方法有哪些？

**答案要点：**

- 均值滤波：简单平滑，但容易模糊边缘。
- 高斯滤波：按高斯权重平滑，常用于降噪。
- 中值滤波：对椒盐噪声效果好。
- 双边滤波：同时考虑空间距离和像素差异，能较好保边。

## 6. Sobel、Laplacian、Canny 有什么区别？

**答案要点：**

- Sobel：一阶梯度算子，分别计算 x 和 y 方向边缘。
- Laplacian：二阶导数算子，对噪声敏感。
- Canny：多阶段边缘检测，包括高斯滤波、梯度计算、非极大值抑制、双阈值连接，效果稳定。

## 7. 什么是直方图均衡化？

**答案要点：**

直方图均衡化通过重新分布像素灰度值，增强图像对比度。

- 适合整体对比度较低的灰度图。
- 对彩色图像通常在亮度通道上处理。
- 可能放大噪声，实际中也可使用自适应直方图均衡化 CLAHE。

## 8. 模板匹配的原理是什么？

**答案要点：**

模板匹配是在目标图像中滑动模板，计算模板与局部区域的相似度，找到最匹配位置。

特点：

- 实现简单。
- 对尺度、旋转、光照变化敏感。
- 适合目标大小和形态稳定的场景。

## 9. 霍夫变换适合检测什么？

**答案要点：**

霍夫变换常用于检测具有参数化形式的几何形状，如直线和圆。

- 图像空间中的点映射到参数空间。
- 参数空间中投票峰值对应可能的几何形状。
- 常结合边缘检测结果使用。

## 10. SIFT、SURF、ORB 有什么区别？

**答案要点：**

- SIFT：尺度不变特征，鲁棒性强，但计算较慢。
- SURF：速度比 SIFT 快，思想相近。
- ORB：结合 FAST 关键点和 BRIEF 描述子，速度快，适合实时场景。
- 面试中可强调特征点检测、描述子提取和特征匹配三个步骤。

## 11. HOG 特征适合什么场景？

**答案要点：**

HOG 统计局部区域梯度方向直方图，用于描述目标轮廓和形状。

经典应用：

- 行人检测。
- 传统目标检测。
- 与 SVM 结合做分类。

## 12. CNN 为什么适合图像任务？

**答案要点：**

CNN 利用图像的局部相关性和空间结构。

- 局部连接减少参数。
- 权重共享提高泛化能力。
- 多层卷积逐步提取从低级到高级的特征。
- 池化或步长卷积降低空间尺寸和计算量。

## 13. AlexNet、VGG、GoogLeNet、ResNet 的核心改进是什么？

**答案要点：**

- AlexNet：使用 ReLU、Dropout、数据增强和 GPU 训练，推动深度 CNN 发展。
- VGG：使用大量小卷积核堆叠，结构规整。
- GoogLeNet：提出 Inception 模块，多尺度特征提取，减少参数。
- ResNet：提出残差连接，解决深层网络退化问题。

## 14. 图像增强有什么作用？

**答案要点：**

图像增强通过对训练图像做随机变换扩充数据分布，提高模型泛化能力。

常见方法：

- 随机裁剪、缩放、翻转、旋转。
- 颜色扰动。
- 随机擦除。
- Mixup、CutMix。

## 15. 目标检测中 two-stage 和 one-stage 有什么区别？

**答案要点：**

- two-stage 方法先生成候选区域，再对候选区域分类和回归，代表算法 Faster R-CNN。
- one-stage 方法直接在特征图上预测类别和边界框，代表算法 YOLO、SSD。
- two-stage 通常精度较高但速度较慢。
- one-stage 通常速度较快，适合实时检测。

## 16. IoU 是什么？

**答案要点：**

IoU 是预测框和真实框交集面积与并集面积的比值。

- 取值范围 0 到 1。
- 用于衡量边界框重叠程度。
- 目标检测评估和 NMS 中都常用。

## 17. NMS 的作用是什么？

**答案要点：**

NMS 即非极大值抑制，用于去除同一目标的重复检测框。

基本流程：

1. 按置信度排序。
2. 选择置信度最高的框。
3. 删除与该框 IoU 超过阈值的其他框。
4. 重复直到没有候选框。

## 18. mAP 是什么？

**答案要点：**

mAP 是目标检测常用评估指标，表示各类别 AP 的平均值。

- AP 是 Precision-Recall 曲线下面积。
- mAP 综合衡量分类和定位效果。
- 不同数据集会使用不同 IoU 阈值，如 VOC 和 COCO 的计算方式不同。

## 19. YOLO 的核心思想是什么？

**答案要点：**

YOLO 将目标检测看作单阶段回归问题，直接从图像特征预测目标类别、边界框和置信度。

特点：

- 检测速度快。
- 适合实时应用。
- 早期版本对小目标和密集目标检测较弱，后续版本不断改进多尺度特征和检测头。

## 20. 语义分割和实例分割有什么区别？

**答案要点：**

- 语义分割为每个像素预测类别，不区分同类不同实例。
- 实例分割既要预测像素类别，又要区分同一类别下不同对象实例。
- UNet 是典型语义分割网络，Mask R-CNN 是典型实例分割方法。

## 21. UNet 的结构特点是什么？

**答案要点：**

UNet 采用编码器-解码器结构，并通过跳跃连接融合浅层细节特征和深层语义特征。

- 编码器负责下采样和语义提取。
- 解码器负责上采样恢复空间分辨率。
- 跳跃连接有助于恢复边界和细节。
- 常用于医学图像分割和小数据分割任务。
