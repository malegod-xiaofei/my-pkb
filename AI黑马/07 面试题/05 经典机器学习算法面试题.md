# 05 经典机器学习算法面试题

> 参考笔记：`AI黑马/03 机器学习算法_经典算法`。
>
> 权威参考：scikit-learn User Guide、The Elements of Statistical Learning、PRML、XGBoost 官方文档、LightGBM 官方文档。

## 1. KNN 算法的基本思想是什么？

**答案要点：**

KNN 的思想是：对一个待预测样本，找到训练集中距离它最近的 K 个样本，根据这些邻居的标签投票或加权投票得到预测结果。

特点：

- 属于懒惰学习，没有显式训练过程。
- 对特征尺度敏感，通常需要归一化或标准化。
- 预测阶段计算成本较高。
- 对异常值、噪声和高维数据较敏感。

## 2. KNN 中 K 值如何选择？

**答案要点：**

- K 太小：模型复杂，容易受噪声影响，方差较高。
- K 太大：模型过于平滑，可能欠拟合，偏差较高。
- 通常通过交叉验证选择 K 值。
- 二分类中 K 可选奇数，减少投票平局。
- 可使用距离加权投票，让更近的邻居权重更大。

## 3. 线性回归的目标函数是什么？

**答案要点：**

线性回归通常最小化预测值与真实值之间的平方误差，即最小二乘损失。

常见求解方式：

- 正规方程：直接求解析解，适合特征维度较低且矩阵可逆的场景。
- 梯度下降：迭代优化，适合大规模数据。
- 随机梯度下降和小批量梯度下降：适合在线或大数据场景。

## 4. L1 正则化和 L2 正则化有什么区别？

**答案要点：**

- L1 正则化倾向于让部分参数变为 0，具有稀疏化和特征选择作用。
- L2 正则化倾向于让参数整体变小但不一定为 0，提升模型稳定性和泛化能力。
- Lasso 使用 L1，Ridge 使用 L2，Elastic Net 同时结合 L1 和 L2。
- 高维稀疏特征中 L1 常用，连续特征中 L2 更平滑稳定。

## 5. 逻辑回归为什么可以用于分类？

**答案要点：**

逻辑回归把线性模型输出通过 Sigmoid 函数映射到 0 到 1 之间，可解释为正类概率。

- 二分类中通常根据阈值判断类别。
- 损失函数常用对数损失或交叉熵。
- 决策边界本质上是线性的，也可以通过特征组合处理非线性问题。
- 优点是简单、可解释、训练快，常作为分类基线模型。

## 6. 逻辑回归和线性回归有什么区别？

**答案要点：**

- 线性回归用于预测连续值，逻辑回归用于分类。
- 线性回归输出任意实数，逻辑回归输出概率。
- 线性回归常用 MSE，逻辑回归常用交叉熵。
- 逻辑回归虽然名字里有回归，但本质是广义线性模型中的分类模型。

## 7. 决策树如何选择划分特征？

**答案要点：**

常见划分准则：

- ID3：信息增益。
- C4.5：信息增益率。
- CART 分类树：基尼指数。
- CART 回归树：平方误差或方差下降。

目标是让划分后的子节点更纯，分类不确定性更低。

## 8. 决策树为什么容易过拟合？如何解决？

**答案要点：**

决策树可以不断划分训练数据，容易把噪声和偶然模式也学习进去。

解决方式：

- 限制最大深度。
- 限制叶子节点最小样本数。
- 限制节点继续划分的最小样本数。
- 剪枝。
- 使用随机森林、GBDT 等集成方法。

## 9. 随机森林的随机性体现在哪里？

**答案要点：**

随机森林是 Bagging 思想下的决策树集成模型，随机性主要体现在：

- 样本随机：每棵树使用 bootstrap 抽样得到训练集。
- 特征随机：每次节点划分只从部分特征中选择最优特征。

这两种随机性可以降低树之间的相关性，从而降低方差。

## 10. Bagging 和 Boosting 有什么区别？

**答案要点：**

- Bagging 并行训练多个基学习器，通过投票或平均降低方差，典型算法是随机森林。
- Boosting 串行训练多个弱学习器，后一个模型重点修正前面模型的错误，主要降低偏差。
- Bagging 对过拟合模型有帮助，Boosting 对弱模型逐步增强。
- Boosting 通常效果强，但对噪声和异常值更敏感。

## 11. GBDT 的核心思想是什么？

**答案要点：**

GBDT 是梯度提升决策树，通过逐步训练多棵回归树来拟合损失函数的负梯度方向。

- 每一轮新树都在修正前面模型的残差或负梯度。
- 最终模型是多棵树的加法模型。
- 适合处理非线性特征和混合类型特征。
- 需要关注学习率、树深度、树数量、采样比例等参数。

## 12. XGBoost 相比传统 GBDT 有哪些改进？

**答案要点：**

- 目标函数加入二阶导数信息。
- 加入正则项控制模型复杂度。
- 支持列采样和行采样。
- 支持缺失值默认方向学习。
- 支持并行化特征分裂查找。
- 工程实现高效，支持缓存优化和分布式训练。

## 13. LightGBM 的主要特点是什么？

**答案要点：**

- 使用直方图算法加速训练。
- 使用 leaf-wise 生长策略，通常能更快降低损失。
- 支持 GOSS 和 EFB 等优化策略。
- 训练速度快、内存占用低，适合大规模数据。
- leaf-wise 更容易过拟合，需要控制叶子数、深度和最小叶子样本数。

## 14. 朴素贝叶斯为什么叫朴素？

**答案要点：**

因为它假设在给定类别的条件下，各特征之间相互独立。这个条件独立假设在现实中往往不完全成立，但可以简化计算，并在文本分类等高维稀疏场景中表现较好。

常见类型：

- 高斯朴素贝叶斯。
- 多项式朴素贝叶斯。
- 伯努利朴素贝叶斯。

## 15. 朴素贝叶斯中的拉普拉斯平滑有什么作用？

**答案要点：**

拉普拉斯平滑用于解决某个特征值在某类别中未出现导致概率为 0 的问题。

- 如果某个条件概率为 0，整个后验概率会变为 0。
- 平滑通过给每个计数加一个较小常数，避免零概率。
- 文本分类中特别常见，因为词表大且稀疏。

## 16. SVM 的核心思想是什么？

**答案要点：**

SVM 的目标是在特征空间中寻找一个分类超平面，使两类样本到超平面的间隔最大。

关键概念：

- 支持向量：决定分类边界的关键样本。
- 最大间隔：提升泛化能力。
- 软间隔：允许少量样本违反间隔约束。
- 核函数：把非线性问题映射到高维空间处理。

## 17. SVM 中参数 C 和 gamma 分别控制什么？

**答案要点：**

- C 控制误分类惩罚强度。C 越大，对训练错误惩罚越大，模型更容易过拟合；C 越小，间隔更宽，模型更简单。
- gamma 常见于 RBF 核，控制单个样本影响范围。gamma 越大，影响范围越小，决策边界更复杂；gamma 越小，边界更平滑。

## 18. KMeans 的基本流程是什么？

**答案要点：**

1. 随机初始化 K 个聚类中心。
2. 将每个样本分配到最近的聚类中心。
3. 根据分配结果重新计算聚类中心。
4. 重复分配和更新，直到中心变化很小或达到迭代次数。

KMeans 对初始点、K 值、异常值和特征尺度都比较敏感。

## 19. 如何选择 KMeans 中的 K 值？

**答案要点：**

常见方法：

- 肘部法：观察簇内平方和随 K 变化的拐点。
- 轮廓系数：衡量样本与本簇和其他簇的距离关系。
- 结合业务含义选择。
- 多次随机初始化，选择效果稳定的结果。

## 20. PCA 的核心思想是什么？

**答案要点：**

PCA 通过线性变换把原始特征投影到方差最大的若干正交方向上，从而实现降维。

- 第一主成分解释最大方差。
- 主成分之间相互正交。
- PCA 是无监督降维方法，不利用标签信息。
- 降维可以去噪、可视化、缓解维灾难，但会损失部分信息。

## 21. EM 算法适合解决什么问题？

**答案要点：**

EM 算法适合含有隐变量或缺失数据的概率模型参数估计。

- E 步：根据当前参数估计隐变量的后验分布或期望。
- M 步：最大化期望完全数据对数似然，更新参数。
- 重复 E 步和 M 步直到收敛。
- 典型应用：高斯混合模型、HMM 参数学习。

## 22. HMM 的三要素和三个基本问题是什么？

**答案要点：**

三要素：

- 初始状态概率。
- 状态转移概率。
- 观测发射概率。

三个基本问题：

- 评估问题：给定模型和观测序列，计算观测序列概率，常用前向后向算法。
- 解码问题：给定模型和观测序列，求最可能隐藏状态序列，常用维特比算法。
- 学习问题：给定观测序列估计模型参数，常用 Baum-Welch 算法。

## 23. 如何对比多个机器学习算法？

**答案要点：**

可以从以下维度回答：

- 模型表达能力。
- 可解释性。
- 对特征尺度是否敏感。
- 对异常值是否敏感。
- 训练和推理复杂度。
- 是否适合高维稀疏特征。
- 对非线性关系的建模能力。
- 工程部署和调参难度。
