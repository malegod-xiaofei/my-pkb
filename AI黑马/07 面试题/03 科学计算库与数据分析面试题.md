# 03 科学计算库与数据分析面试题

> 参考笔记：`AI黑马/02 机器学习基础_科学计算库`。
>
> 权威参考：NumPy、Pandas、Matplotlib、Seaborn 官方文档，以及《Python for Data Analysis》。

## 1. NumPy 的 `ndarray` 和 Python 原生 list 有什么区别？

**答案要点：**

- `ndarray` 通常存储同一类型数据，内存连续，适合向量化计算。
- Python list 存储的是对象引用，可以混合类型，但数值计算效率较低。
- NumPy 底层大量操作由 C 实现，避免 Python 层循环开销。
- `ndarray` 支持广播、矩阵运算、布尔索引、批量聚合等高效操作。

## 2. NumPy 为什么比 Python 循环快？

**答案要点：**

- NumPy 使用连续内存和固定 dtype，CPU 缓存友好。
- 大量操作在 C 层实现，减少 Python 解释器循环开销。
- 向量化操作可以利用底层 BLAS、SIMD 等优化。
- 避免逐元素 Python 对象创建和动态类型检查。

## 3. 什么是广播机制？

**答案要点：**

广播机制允许不同形状的数组在满足规则时进行逐元素运算。

规则：

- 从尾部维度开始对齐。
- 两个维度相等，或者其中一个为 1，则可以广播。
- 不满足上述条件会报 shape 不兼容错误。

广播可以减少显式复制数据，提高计算效率。

## 4. NumPy 中视图和拷贝有什么区别？

**答案要点：**

- 视图共享底层数据，修改视图可能影响原数组。
- 拷贝拥有独立数据，修改拷贝不会影响原数组。
- 普通切片通常返回视图。
- 花式索引、布尔索引通常返回拷贝。
- 面试中可以通过 `base` 属性或修改结果验证是否共享数据。

## 5. Pandas 中 Series 和 DataFrame 有什么区别？

**答案要点：**

- Series 是一维带标签数组。
- DataFrame 是二维表格结构，由多个 Series 组成。
- Series 适合表示单列数据，DataFrame 适合表示结构化数据表。
- 二者都有索引，支持按标签和位置访问。

## 6. `loc` 和 `iloc` 有什么区别？

**答案要点：**

- `loc` 基于标签选择数据。
- `iloc` 基于整数位置选择数据。
- `loc` 的切片通常包含结束标签。
- `iloc` 的切片与 Python 普通切片一致，左闭右开。

## 7. Pandas 如何处理缺失值？

**答案要点：**

常用方法：

- `isna()` / `notna()` 检测缺失值。
- `dropna()` 删除缺失值。
- `fillna()` 填充缺失值。
- 数值特征可用均值、中位数、分组统计值填充。
- 类别特征可用众数、特殊类别填充。

面试中应强调：缺失值处理不能泄漏测试集信息，应只在训练集上拟合填充值。

## 8. `groupby` 的作用是什么？

**答案要点：**

`groupby` 用于按某些字段分组，再对每组执行聚合、转换或过滤。

常见操作：

- `agg()`：聚合统计，如均值、最大值、计数。
- `transform()`：返回与原数据等长的结果。
- `apply()`：自定义处理，但性能通常不如内置向量化操作。

## 9. `merge`、`join`、`concat` 有什么区别？

**答案要点：**

- `merge` 类似 SQL join，按列或索引关联两张表。
- `join` 更偏向按索引连接。
- `concat` 用于沿行或列方向拼接多个对象。
- 常见 join 类型：inner、left、right、outer。

## 10. 什么是透视表？适合什么场景？

**答案要点：**

透视表用于按照行、列维度对数据进行交叉汇总。

适合场景：

- 按地区和月份统计销售额。
- 按用户类型和商品类别统计转化率。
- 对多维指标做聚合分析。

Pandas 中常用 `pivot_table()`。

## 11. 数据分析的一般流程是什么？

**答案要点：**

1. 明确业务问题和分析目标。
2. 获取数据。
3. 数据理解和探索性分析。
4. 数据清洗：缺失值、重复值、异常值、类型转换。
5. 特征构造和统计分析。
6. 可视化展示和结论解释。
7. 输出报告或支撑建模。

## 12. 如何识别和处理异常值？

**答案要点：**

识别方法：

- 箱线图和 IQR。
- Z-score。
- 分位数阈值。
- 业务规则。

处理方法：

- 删除异常样本。
- 截尾或缩尾。
- 替换为合理值。
- 单独作为异常类别或异常标记特征。

处理前应判断异常值是错误数据还是有意义的极端样本。

## 13. Matplotlib 中 Figure 和 Axes 有什么区别？

**答案要点：**

- Figure 是整张画布。
- Axes 是具体绘图区，一个 Figure 可以包含多个 Axes。
- 面向对象 API 更适合复杂图表布局。
- `pyplot` API 简单方便，适合快速绘图。

## 14. 常见可视化图形分别适合什么场景？

**答案要点：**

- 折线图：展示趋势变化。
- 柱状图：比较不同类别数值。
- 散点图：观察两个变量关系。
- 直方图：观察数据分布。
- 箱线图：观察分布、离散程度和异常值。
- 热力图：展示矩阵、相关系数或二维强度。

## 15. 如何做探索性数据分析？

**答案要点：**

- 查看数据规模、字段类型、缺失率、重复率。
- 计算描述性统计指标。
- 分析单变量分布。
- 分析变量之间相关关系。
- 对目标变量进行分组对比。
- 结合可视化寻找异常、趋势和业务规律。

## 16. 如果一个 DataFrame 很大，Pandas 处理很慢怎么办？

**答案要点：**

- 优先使用向量化操作，避免逐行循环。
- 优化 dtype，例如把字符串类别转为 category。
- 只读取需要的列。
- 分块读取和处理。
- 使用索引优化查询。
- 对超大数据可考虑 Dask、Spark、数据库或 Polars。

## 17. 数据泄漏在数据分析和建模中如何产生？

**答案要点：**

数据泄漏是训练过程使用了预测时不可获得的信息。

常见场景：

- 先对全量数据标准化，再划分训练集和测试集。
- 特征中包含目标变量的后验信息。
- 时间序列任务中使用了未来数据。
- 交叉验证时在折外数据上拟合了预处理器。

正确做法是把预处理步骤放入训练流程，只在训练集上拟合。

## 18. 北京租房或 NBA 数据分析项目面试时怎么介绍？

**答案要点：**

可以按以下结构回答：

1. 项目目标：分析租金影响因素或球员表现规律。
2. 数据来源和字段说明。
3. 数据清洗：缺失值、异常值、重复值、类型转换。
4. 特征分析：地区、面积、户型、位置、球队、得分等。
5. 可视化：分布、趋势、相关性、分组对比。
6. 结论：影响因素、业务建议、后续建模方向。
