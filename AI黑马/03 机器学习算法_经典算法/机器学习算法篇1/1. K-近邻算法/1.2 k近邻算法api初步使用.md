### 1.1 安装
```plain
pip3 install scikit-learn==0.19.1
```

安装好之后可以通过以下命令查看是否安装成功

```plain
import sklearn
```

+ 注：安装scikit-learn需要Numpy, Scipy等库

### 3.2 代码过程
+ 导入模块 

```plain
from sklearn.neighbors import KNeighborsClassifier
```

+ 构造数据集

```plain
x = [[0], [1], [2], [3]]
y = [0, 0, 1, 1]
```

+ 机器学习 -- 模型训练

```plain
# 实例化API
estimator = KNeighborsClassifier(n_neighbors=2)
# 使用fit方法进行训练
estimator.fit(x, y)

estimator.predict([[1]])
```
