## 查看目录命令
**学习目标**

+ 能够使用Linux命令查看目录信息

---

### 1. 查看目录命令的使用
| 命令 | 说明 |
| :--- | :--- |
| ls | 查看当前目录信息 |
| tree | 以树状方式显示目录信息 |

### 4. 小结
+ ls和tree是我们查看目录信息最常用的命令
+ ls只能查看一层目录信息
+ tree可以查看多层目录信息。
