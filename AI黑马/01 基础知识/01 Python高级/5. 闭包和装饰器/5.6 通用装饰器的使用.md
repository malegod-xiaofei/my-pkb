## 通用装饰器的使用
**学习目标**

+ 能够写出通用的装饰器

前面装饰器只能装饰没有参数没有返回值的函数,适用范围不广泛,如何装饰带有参数以及返回值的函数呢?

---

### 1. 装饰带有参数的函数
```plain
# 添加输出日志的功能
def logging(fn):
    def inner(num1, num2):
        print("--正在努力计算--")
        fn(num1, num2)

    return inner


# 使用装饰器装饰函数
@logging
def sum_num(a, b):
    result = a + b
    print(result)


sum_num(1, 2)
```

**运行结果:**

```plain
--正在努力计算--
3
```

> 装饰带有参数的函数时,需要在装饰器内部函数(inner)中定义参数,并且调用函数参数(fn)时传递参数
>

### 2. 装饰带有返回值的函数
```plain
# 添加输出日志的功能
def logging(fn):
    def inner(num1, num2):
        print("--正在努力计算--")
        result = fn(num1, num2)
        return result
    return inner


# 使用装饰器装饰函数
@logging
def sum_num(a, b):
    result = a + b
    return result


result = sum_num(1, 2)
print(result)
```

**运行结果:**

```plain
--正在努力计算--
3
```

> 装饰有返回值的函数需要在内部函数(inner)中返回调用函数参数(fn)的返回值
>

### 3. 装饰带有不定长参数的函数
```plain
# 添加输出日志的功能
def logging(fn):
    def inner(*args, **kwargs):
        print("--正在努力计算--")
        fn(*args, **kwargs)

    return inner


# 使用语法糖装饰函数
@logging
def sum_num(*args, **kwargs):
    result = 0
    for value in args:
        result += value

    for value in kwargs.values():
        result += value

    print(result)

sum_num(1, 2, a=10)
```

**运行结果:**

```plain
--正在努力计算--
13
```

### 4. 通用装饰器
```plain
# 添加输出日志的功能
def logging(fn):
    def inner(*args, **kwargs):
        print("--正在努力计算--")
        result = fn(*args, **kwargs)
        return result

    return inner


# 使用语法糖装饰函数
@logging
def sum_num(*args, **kwargs):
    result = 0
    for value in args:
        result += value

    for value in kwargs.values():
        result += value

    return result


@logging
def subtraction(a, b):
    result = a - b
    print(result)

result = sum_num(1, 2, a=10)
print(result)

subtraction(4, 2)
```

**运行结果:**

```plain
--正在努力计算--
13
--正在努力计算--
2
```

### 5. 小结
+ 通用装饰器的语法格式:

```plain
# 通用装饰器
def logging(fn):
  def inner(*args, **kwargs):
      print("--正在努力计算--")
      result = fn(*args, **kwargs)
      return result

  return inner
```
