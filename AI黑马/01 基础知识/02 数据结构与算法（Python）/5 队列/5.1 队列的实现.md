## 队列的实现
同栈一样，队列也可以用顺序表或者链表实现。

### 操作
+ Queue() 创建一个空的队列
+ enqueue(item) 往队列中添加一个item元素
+ dequeue() 从队列头部删除一个元素
+ is_empty() 判断一个队列是否为空
+ size() 返回队列的大小

### 顺序表实现队列
+ 列表实现分析

**头部插入删除数据**: insert(0,item)时间复杂度: O(n) pop(0)时间复杂度: O(n) **尾部插入删除数据**: append(item)时间复杂度: O(1) pop()时间复杂度: O(1)

> 头部插入数据尾部删除数据或者尾部插入数据头部删除数据时间复杂度一样
>
> 如下代码采用头部增加数据,尾部删除数据方式实现
>

+ 代码实现

```plain
class Queue(object):
    """队列"""
    def __init__(self):
        self.items = []

    def is_empty(self):
        return self.items == []

    def enqueue(self, item):
        """进队列"""
        self.items.insert(0,item)

    def dequeue(self):
        """出队列"""
        return self.items.pop()

    def size(self):
        """返回大小"""
        return len(self.items)

if __name__ == "__main__":
    q = Queue()
    q.enqueue("hello")
    q.enqueue("world")
    q.enqueue("itcast")
    print q.size()
    print q.dequeue()
    print q.dequeue()
    print q.dequeue()
```

### 链表实现队列
+ 链表实现分析

**链表特殊处理**

对于栈来说，我们只需要一个**栈顶指针**就可以了。但是队列需要两个指针：一个是head指针，指向队头；一个是tail指针，指向队尾。

![](https://cdn.nlark.com/yuque/0/2024/jpeg/25755350/1730458000708-956a9cba-9335-4f9b-bafa-975d5de501b3.jpeg)

这样做可以直接获得尾部节点,好处是可以把尾部操作的时间复杂度从O(n)降低到O(1)

**插入和删除端**

采用的从尾部增加数据,头部删除数据

因为,如果尾部删除数据的话,tail需要指向尾部上一个节点,上一个节点是拿不到的

+ 代码实现

```plain
class Node:
    def __init__(self, data, next=None):
        self.data = data
        self.next = next

class LinkedQueue:
    def __init__(self):
        self.__head = None
        self.__tail = None

    def enqueue(self, value):
        new_node = Node(value)
        if self.__tail:
            self.__tail.next = new_node
        else:
            self.__head = new_node
        self.__tail = new_node

    def dequeue(self):
        if self.__head:
            value = self.__head.data
            self.__head = self.__head.next
            if not self.__head:
                self.__tail = None
            return value
```
