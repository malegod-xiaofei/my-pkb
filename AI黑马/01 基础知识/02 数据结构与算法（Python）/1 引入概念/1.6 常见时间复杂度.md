**课堂练习**

下面程序时间复杂度:

```plain
def cal(m, n):
    sum_1 = 0
    for i in range(1, m):
        sum_1 += i

    sum_2 = 0
    for j in range(1, n):
        sum_2 += j
    return sum_1 + sum_2
```

O(m+n)
