# 1 基础操作

## 1.1 python 下载指定版本的 jar 包

```bash
pip install elasticsearch -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
D:\python3\python.exe -m pip install numpy -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
D:\Software\Python37\python.exe -m pip install Scikit-learn -i http://pypi.douban.com/simple/ --trusted-host pypi.douban.com
```

## 1.2 python打包

首先进入到当前待打包目录下，-F参数打包指定的文件，打包完成后就会在dist下生成：文件名.exe

```bash
pyinstaller -F platform_province_4_ES_TO_DWS.py
```

## 1.3 升级pip版本

```bash
python -m pip install --upgrade pip
pip uninstall pyzmq # 卸载pyzmq库
pip install pyzmq==19.0.2 # 下载19.0.2这个版本的pyzmq库
```
