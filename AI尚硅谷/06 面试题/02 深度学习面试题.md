## 1. 神经网络的基本结构是什么？

**答案要点：**

神经网络通常由输入层、隐藏层和输出层组成。

- 输入层：接收原始特征。
- 隐藏层：通过权重、偏置和激活函数提取非线性特征。
- 输出层：输出分类概率、回归值或其他任务结果。

每个神经元一般执行“加权求和 + 偏置 + 激活函数”的计算。

## 2. 为什么神经网络需要激活函数？

**答案要点：**

激活函数的核心作用是引入非线性。如果没有激活函数，多层线性变换仍然可以合并为一个线性变换，深层网络就失去了表达复杂非线性关系的能力。

常见激活函数：

- Sigmoid：输出在 0 到 1 之间，早期常用，但容易梯度消失。
- Tanh：输出在 -1 到 1 之间，零中心，但也可能梯度消失。
- ReLU：计算简单，收敛快，深度学习中常用。
- GELU / SiLU：大模型中较常见。

## 3. ReLU 有什么优缺点？

**答案要点：**

优点：

- 计算简单。
- 正区间梯度恒定，有助于缓解梯度消失。
- 输出具有稀疏性，提升计算效率。

缺点：

- 当输入长期为负时，输出为 0，梯度也为 0，可能出现“神经元死亡”。

常见改进：Leaky ReLU、PReLU、ELU、GELU。

## 4. Softmax 的作用是什么？为什么要做数值稳定处理？

**答案要点：**

Softmax 将一组实数转换为概率分布，常用于多分类输出层。

为了避免指数计算溢出，通常会先减去最大值：

```python
x = x - max(x)
softmax = exp(x) / sum(exp(x))
```

这种处理不会改变最终概率分布，但能显著提升数值稳定性。

## 5. 什么是前向传播和反向传播？

**答案要点：**

- 前向传播：输入数据逐层经过网络计算，得到预测结果和损失。
- 反向传播：根据链式法则，从输出层向前逐层计算损失对参数的梯度。
- 参数更新：优化器根据梯度更新权重和偏置。

一句话总结：前向传播算结果，反向传播算梯度，优化器负责更新参数。

## 6. 什么是梯度消失和梯度爆炸？如何解决？

**答案要点：**

- 梯度消失：反向传播过程中梯度越来越小，前面层几乎学不到东西。
- 梯度爆炸：梯度越来越大，导致训练不稳定甚至数值溢出。

常见解决方式：

- 使用 ReLU、GELU 等激活函数。
- 合理初始化参数。
- 使用 BatchNorm、LayerNorm、RMSNorm。
- 使用残差连接。
- 梯度裁剪。
- 调整学习率。

## 7. BatchNorm 和 LayerNorm 有什么区别？

**答案要点：**

- BatchNorm：在 batch 维度统计均值和方差，常用于 CNN，对 batch size 较敏感。
- LayerNorm：在单个样本的特征维度上归一化，与 batch size 无关，常用于 RNN 和 Transformer。
- 大模型中常见 LayerNorm、RMSNorm，因为训练和推理时对 batch 大小依赖更小。

## 8. Dropout 的作用是什么？

**答案要点：**

Dropout 在训练时随机丢弃部分神经元，使网络不能过度依赖某些局部特征，从而缓解过拟合。

- 训练时启用 Dropout。
- 推理时关闭 Dropout，并使用完整网络。
- 常用于全连接层、Transformer 层等位置。

## 9. 什么是残差连接？为什么有用？

**答案要点：**

残差连接将某层输入直接加到该层或后续层的输出上。

作用：

- 为梯度提供更短的传播路径。
- 缓解深层网络训练困难。
- 让模型学习“增量变化”，而不是从零学习完整映射。
- ResNet 和 Transformer 都大量使用残差连接。

## 10. CNN 的核心思想是什么？

**答案要点：**

CNN 的核心是卷积操作，适合处理图像等具有局部结构的数据。

关键特点：

- 局部感受野：卷积核只关注局部区域。
- 权重共享：同一个卷积核在不同位置复用参数。
- 池化：降低空间尺寸，增强平移不变性。

CNN 常用于图像分类、目标检测、图像分割等任务。

## 11. RNN、LSTM、GRU 有什么区别？

**答案要点：**

- RNN：通过隐藏状态处理序列，但容易梯度消失，长距离依赖建模较弱。
- LSTM：引入输入门、遗忘门、输出门和细胞状态，缓解长期依赖问题。
- GRU：结构比 LSTM 更简单，参数更少，训练效率更高。

Transformer 出现后，在多数 NLP 场景中逐渐替代传统 RNN 系列结构。

## 12. Adam 优化器为什么常用？AdamW 又改进了什么？

**答案要点：**

Adam 结合了 Momentum 和 RMSProp 的思想：

- 一阶矩估计：类似动量，平滑梯度方向。
- 二阶矩估计：自适应调整每个参数的学习率。

AdamW 的主要改进是解耦权重衰减，使正则化行为更符合预期，在 Transformer 和大模型训练中更常见。

## 13. 学习率过大或过小会怎样？

**答案要点：**

- 学习率过大：训练震荡，损失不收敛，甚至发散。
- 学习率过小：收敛很慢，可能陷入局部区域。

常用策略：

- Warmup：训练初期逐步增大学习率。
- Step Decay：按阶段衰减。
- Cosine Decay：余弦退火。
- ReduceLROnPlateau：验证集不提升时降低学习率。

## 14. 什么是混合精度训练？为什么大模型常用？

**答案要点：**

混合精度训练使用 FP16、BF16 等低精度格式进行大部分计算，同时保留关键部分的数值稳定性。

优点：

- 降低显存占用。
- 提升训练速度。
- 支持更大 batch size 或更大模型。

BF16 动态范围更大，在大模型训练中比 FP16 更稳定。

## 15. PyTorch 中训练一个模型通常有哪些步骤？

**答案要点：**

1. 准备 Dataset 和 DataLoader。
2. 定义模型 `nn.Module`。
3. 定义损失函数。
4. 定义优化器。
5. 训练循环：前向传播、计算损失、清空梯度、反向传播、参数更新。
6. 验证和保存模型。

典型代码流程：

```python
optimizer.zero_grad()
y_pred = model(x)
loss = criterion(y_pred, y)
loss.backward()
optimizer.step()
```
