# Dify介绍

Dify(Do It For You)：是一个用于构建 AI 工作流的开源平台。通过在可视化画布上编排AI模型、连接数据源、定义处理流程，直接将领域知识转换为可运行的软件。

# 特点

优点：开源，可本地部署，数据安全。

缺点：插件国际化，需要魔法下载

  

本质就是一个规则引擎，支持动态的安装插件、支持节点中编写代码。

# 搭建

## Docker方式搭建

环境准备：安装docker、docker-compose

  

1. 从DIfy的Github仓库中拉取源码
2. 将源码中的 `.env.example` 文件复制出一份`.env`文件，并按自己的需要调整里面的内容（例如nginx端口号等）
3. 运行源码中的docker compose配置文件

```
docker compose up -d
```

1. 启动完成后，可以通过浏览器进行验证：[http://127.0.0.1:80/](http://127.0.0.1:80/)
2. 首次登录时，需要设置管理员的邮箱、用户名、密码

# 导入导出工作流

可以将Dify中创建好的工作流进行导出：选中要导出的工作流，选择`导出DSL`，会生成一个 yml 文件导出出来。

  

在Dify首页，可以选择`导入DSL文件`，将要导入的工作流yml文件进行导入。
